# Answer Key — Demo Quiz 001

Quiz: Cambridge IGCSE Physics 0625 — Demo Quiz 001

---

## Question 1

**Correct answer:** B

**Explanation:** The number of neutrons is found by subtracting the proton number from the nucleon number: 108 - 47 = 61.

**Common misconception:** Students often confuse nucleon number with neutron number and give the nucleon number itself as the answer, forgetting that nucleon number = protons + neutrons.

---

## Question 2

**Correct answer:** D

**Explanation:** Nucleon number is the total of protons and neutrons: 19 + 21 = 40.

**Common misconception:** Some students subtract the proton number from the neutron number instead of adding them, since they mix up which quantities should be combined.

---

## Question 3

**Correct answer:** C

**Explanation:** Neutrons in isotope 1 = 63 - 29 = 34. Neutrons in isotope 2 = 65 - 29 = 36. The difference is 36 - 34 = 2.

**Common misconception:** Students sometimes think isotopes differ in proton number rather than neutron number, leading them to answer 0 because they assume the nuclei are otherwise identical in every count.

---

## Question 4

**Correct answer:** B

**Explanation:** In a neutral atom, the number of electrons equals the number of protons, so the proton number is 34. Neutrons = nucleon number - proton number = 79 - 34 = 45.

**Common misconception:** Students may forget that electron number equals proton number only in a neutral atom, and may instead try to use the electron number directly as a separate quantity unrelated to the proton count.

---

## Question 5

**Correct answer:** C

**Explanation:** An alpha particle carries away a nucleon number of 4, so the original nucleus had a nucleon number of 206 + 4 = 210. Its neutron number is therefore 210 - 84 = 126.

**Common misconception:** Students often forget to add back the 4 nucleons lost as the alpha particle before subtracting the proton number, leading them to use the daughter nucleus's nucleon number directly.
